# MythOS Color System

Version: 0.1


# 1. Introduction

Colors in MythOS are functional.

Every color communicates meaning.


# 2. Primary Colors


## Void Black

HEX:

#0D0E15


Usage:

- Main background
- Desktop environment
- Application surfaces


## Myth Cyan

HEX:

#00E5FF


Usage:

- Active states
- Selection
- Information
- System highlights


## Myth Purple

HEX:

#8B5CFF


Usage:

- AI features
- Intelligent actions
- Advanced operations


# 3. Text Colors


## Primary Text

HEX:

#F5F7FF


Usage:

Main content


## Secondary Text

HEX:

#9CA3AF


Usage:

Secondary information


# 4. Semantic Colors


## Success

Green

Usage:

- Completed actions
- Healthy system states


## Warning

Orange

Usage:

- Attention
- Important notifications


## Error

Red

Usage:

- Failures
- Critical problems


# 5. Glow Rules


Glow must communicate state.

Glow should never exist only as decoration.


Cyan glow:

Active


Purple glow:

AI


Green glow:

Success


Red glow:

Error


# 6. Contrast

All interfaces must maintain:

- High readability
- Clear hierarchy
- Accessibility
