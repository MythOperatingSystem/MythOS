# MythOS Typography

Version: 0.1


# 1. Introduction

Typography defines readability and personality.

MythOS uses a modern technical visual language.


# 2. UI Font


Primary Font:

Inter


Usage:

- Applications
- Menus
- Settings
- System interfaces


Reason:

- Modern
- Clean
- Highly readable


# 3. Code Font


Primary Code Font:

JetBrains Mono


Usage:

- Terminal
- Code editor
- Developer tools


Reason:

- Developer focused
- Clear character distinction


# 4. Typography Scale


## Display

Large system titles


## Heading

Application sections


## Body

Main content


## Caption

Secondary information


# 5. Rules


Typography must be:

- Clean
- Consistent
- Easy to scan


Avoid:

- Excessive font variations
- Decorative fonts
